What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (such as an HTML or XML file) as a tree of objects that can be accessed, modified, and manipulated by programming languages like JavaScript. This allows developers to dynamically change the content, structure, and styling of a web page.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events for its child elements, even if those child elements are added dynamically. Instead of adding event listeners to multiple child elements individually, you use the parent element to listen for events that "bubble up" from the children.
